# AGENTS

This repo is a **minimal RL agent** template.

## Rule

Keep the repo **small** and **educational**.

Do **not** add new algorithms, dashboards, experiment trackers, Docker, CI, or cloud tooling unless explicitly requested.

## Teaching goal

Every file should map back to the loop:

**Observe → Think → Decide → Act → Learn → Repeat**

## Style

- readable code
- simple comments
- no unnecessary abstraction
- beginner-friendly but technically honest
